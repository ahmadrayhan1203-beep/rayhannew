Jelaskan !* Skema jaringan yang paling tepat adalah **Virtual Private Server (VPS)* atau *Shared Hosting* dengan menggunakan *Virtualisasi Jaringan (seperti VLAN atau Subnetting). * *Penjelasan:* * Menggunakan VPS akan memungkinkan Anda membagi satu server fisik menjadi beberapa server virtual yang masing-masing dapat memiliki IP Address publik sendiri dari range yang Anda miliki. * Anda dapat menggunakan *Subnetting* pada IP utama $10.10.1.1/8$ untuk membuat range IP yang spesifik bagi 10 klien dan memisahkannya dari jaringan manajemen server utama. * Jika menggunakan Shared Hosting, setiap klien akan berbagi satu IP Address, tetapi diatur menggunakan virtual host pada web server (misalnya Apache atau Nginx). 37. *Jelaskan perbedaan antara shared hosting dan dedicated hosting!* | Fitur | Shared Hosting | Dedicated Hosting | | :--- | :--- | :--- | | *Sumber Daya* | [cite_start]Berbagi sumber daya (CPU, RAM, disk space) dengan banyak pengguna lain. [cite: 2] | [cite_start]Sumber daya penuh dari satu server fisik digunakan hanya oleh satu pengguna. [cite: 3] | | *Kinerja* | Kinerja dipengaruhi oleh lalu lintas atau aktivitas pengguna lain (noisy neighbor). | Kinerja stabil dan maksimal, tidak dipengaruhi pengguna lain. | | *Kontrol* | Kontrol terbatas, hanya bisa mengelola aplikasi/file website melalui control panel. | [cite_start]Kontrol penuh (root access) untuk mengelola dan mengatur konfigurasi server secara mandiri. [cite: 3] | | *Keamanan* | Risiko keamanan lebih tinggi karena berbagi lingkungan dengan pengguna lain. | Keamanan lebih terjamin karena server hanya dikelola oleh satu pengguna. | | *Harga* | Relatif murah. | Mahal. | 38. *Jelaskan alasan VPS lebih manageable dibanding shared hosting!* * *Kontrol Penuh (*Root Access):* VPS memberikan akses *root atau administrator, yang memungkinkan pengguna untuk menginstal, mengkonfigurasi, dan memodifikasi perangkat lunak atau pengaturan sistem operasi sesuai kebutuhan. [cite_start]Hal ini tidak mungkin dilakukan pada shared hosting[cite: 3]. * *Isolasi Sumber Daya:* Sumber daya (RAM, CPU, disk space) pada VPS dialokasikan secara eksklusif, sehingga kinerja tidak dipengaruhi oleh aktivitas pengguna lain, membuatnya lebih mudah dikelola dan diprediksi. * *Kustomisasi:* Pengguna dapat memilih dan menginstal sistem operasi, web server, database, dan control panel sesuai dengan kebutuhan spesifik aplikasi mereka, memberikan manageability yang lebih tinggi. 39. *Hendra adalah seorang staf IT di perusahaan ABC Samarinda. Jelaskan alasannya!* * *Jenis Hosting:* *Dedicated Hosting* atau *Virtual Private Server (VPS)*. * *Alasan:* * [cite_start]Sistem e-commerce yang terkoneksi dengan sistem keuangan perusahaan dan perbankan membutuhkan *keamanan tingkat tinggi* dan *kinerja yang stabil*[cite: 3]. * [cite_start]*Dedicated Hosting* (atau VPS dengan spesifikasi tinggi) memastikan *sumber daya eksklusif, **isolasi keamanan, dan **kontrol penuh* atas konfigurasi server untuk memenuhi standar regulasi keuangan dan menangani transaksi yang sensitif[cite: 3]. * [cite_start]Shared hosting tidak disarankan karena kurang aman, sumber dayanya terbatas, dan tidak memberikan kontrol penuh yang diperlukan untuk integrasi sistem kritikal seperti keuangan dan perbankan[cite: 2]. 40. *Seorang siswa yang ingin memiliki web pribadi yang dapat menampilkan profil datanya dan beberapa catatan tentang mata pelajaran yang pernah diikutinya.
