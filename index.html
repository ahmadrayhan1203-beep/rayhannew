II. Soal Esai

36. Skema Jaringan untuk Usaha Penyewaan Hosting

Skema jaringan yang paling tepat adalah Virtual Private Server (VPS) atau Shared Hosting dengan menggunakan Virtualisasi Jaringan (Subnetting/VLAN).

Penjelasan:

  • VPS: Memungkinkan pembagian satu server fisik menjadi server-server virtual terisolasi, di mana masing-masing dapat dialokasikan IP Address publik dari range yang tersedia.
  • Subnetting: Digunakan untuk membuat range IP spesifik (10 IP) bagi 10 klien, memisahkannya dari jaringan manajemen server utama ($10.10.1.1/8$) untuk meningkatkan keamanan dan pengelolaan.

37. Perbedaan antara Shared Hosting dan Dedicated Hosting

Fitur Shared Hosting ü§ù Dedicated Hosting üëë
Sumber Daya Berbagi (CPU, RAM, Disk Space) dengan banyak pengguna lain. Sumber daya penuh dari satu server fisik digunakan hanya oleh satu pengguna.
Kontrol Terbatas, hanya mengelola aplikasi/file via control panel. Penuh (Root Access) untuk mengelola seluruh konfigurasi server.
Harga Relatif **murah** dan terjangkau. Relatif **mahal** (investasi penuh).

38. Alasan VPS Lebih Manageable Dibanding Shared Hosting

VPS lebih mudah dikelola karena:

  • Kontrol Penuh (*Root Access*): Pengguna dapat menginstal, mengkonfigurasi, dan memodifikasi perangkat lunak/sistem operasi sesuai kebutuhan, hal yang tidak mungkin pada *shared hosting*.
  • Isolasi Sumber Daya: Kinerja dijamin stabil karena sumber daya dialokasikan secara eksklusif dan tidak dipengaruhi oleh aktivitas pengguna lain.
  • Kustomisasi: Fleksibilitas untuk memilih OS, *web server*, dan *security patch* yang spesifik.

39. Jenis Hosting untuk Sistem E-Commerce yang Terkoneksi dengan Keuangan

Jenis Hosting: Dedicated Hosting atau Virtual Private Server (VPS) Tingkat Tinggi.

Alasan:

  • Sistem yang terintegrasi dengan keuangan dan perbankan membutuhkan keamanan tingkat tinggi dan kinerja stabil.
  • Dedicated Hosting/VPS menawarkan sumber daya eksklusif, isolasi keamanan, dan kontrol penuh (*root access*) untuk memenuhi standar regulasi dan menangani transaksi sensitif.

40. Saran Hosting untuk Siswa (Web Pribadi)

Saran yang diberikan kepada siswa:

  • Pilihan Hosting: Pilih Shared Hosting atau layanan *blogging* gratis (Blogger/WordPress.com) karena murah dan sumber daya yang dibutuhkan rendah.
  • Pilihan Domain: Gunakan domain gratis (seperti .tk) atau *subdomain* dari layanan blog gratis.
  • CMS: Disarankan menggunakan WordPress karena mudah diinstal dan dikelola, cocok untuk pemula.
