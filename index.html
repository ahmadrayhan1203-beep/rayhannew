II. Soal Esai (ASJ)

36. Skema Jaringan untuk Usaha Penyewaan Hosting

Skema jaringan yang paling tepat adalah **Virtual Private Server (VPS)** atau **Shared Hosting** dengan menggunakan **Virtualisasi Jaringan (Subnetting/VLAN)**.

  • **VPS:** Memungkinkan pembagian satu server fisik menjadi server-server virtual terisolasi.
  • **Subnetting:** Digunakan untuk membuat range IP spesifik (10 IP) bagi 10 klien ($10.10.1.1/8$ hingga $10.10.1.10/8$).

37. Perbedaan antara Shared Hosting dan Dedicated Hosting

FiturShared Hosting ü§ùDedicated Hosting üëë
Sumber DayaBerbagi (CPU, RAM, Disk Space) dengan banyak pengguna lain.Sumber daya penuh dari satu server fisik digunakan hanya oleh satu pengguna.
KontrolTerbatas, hanya mengelola aplikasi/file via control panel.Penuh (Root Access) untuk mengelola seluruh konfigurasi server.
HargaRelatif **murah** dan terjangkau.Relatif **mahal** (investasi penuh).

38. Alasan VPS Lebih Manageable Dibanding Shared Hosting

VPS lebih mudah dikelola karena memiliki **Kontrol Penuh (*Root Access*)**, **Isolasi Sumber Daya** yang menjamin kinerja stabil, dan **Kustomisasi** yang fleksibel untuk OS/aplikasi.

39. Jenis Hosting untuk Sistem E-Commerce yang Terkoneksi dengan Keuangan

Jenis Hosting: **Dedicated Hosting** atau **Virtual Private Server (VPS) Tingkat Tinggi**.

Alasan utamanya adalah kebutuhan akan **keamanan tingkat tinggi**, **kinerja stabil**, dan **kontrol penuh** untuk mengelola data dan transaksi sensitif.

40. Saran Hosting untuk Siswa (Web Pribadi)

Pilih **Shared Hosting** atau layanan *blogging* gratis (Blogger/WordPress.com) dengan **domain gratis** (.tk) atau *subdomain*, serta menggunakan **WordPress** sebagai CMS karena mudah dikelola oleh pemula.
